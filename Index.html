--- 🎨 style.css (COPIÁ ESTE CÓDIGO) body { font-family: sans-serif; text-align: center; background-color: #f7f7f7; } h1 { margin-top: 20px; color: #1e3a8a; } .malla { display: flex; flex-wrap: wrap; justify-content: center; gap: 20px; margin: 30px; } .semestre { background-color: #ffffff; border: 2px solid #ccc; border-radius: 10px; padding: 15px; width: 250px; } .semestre h2 { font-size: 18px; color: #333; } .materia { background-color: #e5e7eb; margin: 8px 0; padding: 10px; border-radius: 8px; cursor: pointer; transition: 0.3s; } /* Colores según estado */ .materia.aprobada { background-color: #16a34a; color: white; } .materia.en-curso { background-color: #f97316; color: white; } .materia.pendiente { background-color: #9ca3af; color: white; } .materia:hover { opacity: 0.85; } --- ⚙️ script.js (interactividad opcional) // Alternar entre estados: pendiente → en curso → aprobada const materias = document.querySelectorAll('.materia'); materias.forEach(m => { m.classList.add('pendiente'); // Estado inicial m.addEventListener('click', () => { if (m.classList.contains('pendiente')) { m.classList.remove('pendiente'); m.classList.add('en-curso'); } else if (m.classList.contains('en-curso')) { m.classList.remove('en-curso'); m.classList.add('aprobada'); } else { m.classList.remove('aprobada'); m.classList.add('pendiente'); } }); }); --- 🔼 3. Subir los archivos a GitHub 1. En tu repositorio, hacé clic en "Add file" → "Upload files" 2. Subí los 3 archivos: index.html, style.css, script.js 3. Clic en "Commit changes" --- 🌐 4. Activar GitHub Pages 1. Andá a la pestaña Settings 2. Buscá la sección "Pages" 3. En "Source", elegí la rama main y carpeta /root 4. Guardá y GitHub te va a dar un link como: https://tuusuario.github.io/malla-biotecnologia/ --- ✅ ¡Listo!
